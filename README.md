## 🎯 Mission

We're calling on **Hermès** to:

1. **End the quota system** (配货制度) - No more forced "matching purchases"
2. **Sell at cost price** - Transform luxury into a public good
3. **Become a social enterprise** - Profits go to charity, not shareholders

> *"A Birkin bag costs only $800 to make, yet sells for $12,000+ with forced upselling. It's time for change."*

## 🚀 Quick Start

### 1. Clone & Run Locally

```bash
git clone https://github.com/harviex/hermes-ethical-petition.git
cd hermes-ethical-petition

# Install dependencies
pip install -r requirements.txt

# Run Flask server
python api/index.py
```

Visit `http://localhost:5000`

### 2. Deploy to Vercel (One-Click)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/harviex/hermes-ethical-petition)

Or manually:

```bash
npm i -g vercel
vercel --prod
```

## 🌍 Multi-Language Support

Automatically detects user's country via IP and switches language:

- 🇨🇳 **中文 (zh)** - Default for China/Taiwan/HK
- 🇺🇸 **English (en)** - Default for US/UK/CA/AU
- 🇫🇷 **Français (fr)** - Default for France
- 🇯🇵 **日本語 (ja)** - Default for Japan
- 🇰🇷 **한국어 (ko)** - Default for South Korea

Manual override available via language switcher.
